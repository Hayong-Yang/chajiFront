# 차지차지 (ChajiChaji) — EV 충전소 맞춤 추천 · Frontend

전기차 충전소를 **실질 전비, 가격, 충전 속도/타입, 사업자** 기준으로 한눈에 비교하고 추천해주는 서비스의 프론트엔드입니다.  
UI/UX 중심으로 지도, 필터, 검색, 즐겨찾기, 마이페이지(내 차 등록)까지 제공합니다.
